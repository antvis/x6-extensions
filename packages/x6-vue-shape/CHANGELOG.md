# 变更日志

## 3.0.2 - 2025-12-01

### 修复

- 修复构建产物目录结构：移除 `lib/src` 与 `es/src`，编译产物仅包含模块文件及声明。

## 3.0.1 - 2025-11-27

### 重大变更

- 对齐 `@antv/x6@^3.x`，不兼容旧版 `x6@^2`。
- 兼容 `vue@^2 || >=3`，在 Vue2 环境需安装 `@vue/composition-api`；引入 `vue-demi` 做桥接。

### 新特性

- 统一的组合式 API 支持，在 X6 v3 下的渲染与传送实现优化。

### 维护

- 构建与打包配置更新：Rollup 3、Vite 7。
- UMD 产物：`dist/x6-vue-shape.min.js`，全局名：`X6VueShape`。
